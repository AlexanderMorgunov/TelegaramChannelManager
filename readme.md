# Telegram Channel Manager

Telegram Channel Manager - это веб-приложение для управления каналами Telegram с возможностью добавления новых каналов через QR-код.

## Описание

Приложение представляет собой веб-интерфейс для управления каналами Telegram. Приложение позволяет:

- Просматривать список каналов
- Добавлять новые каналы
- Удалять существующие каналы
- Искать каналы по различным параметрам
- Редактировать настройки каналов
- Генерировать QR-коды для авторизации

## Технологический стек

- **HTML5** - Структура и разметка
- **CSS3** - Стили и адаптивность
- **JavaScript (ES6+)** - Логика и интерактивность
- **Canvas API** - Генерация QR-кодов

## Функциональность

### Основные возможности:

1. **Управление каналами**

   - Отображение списка каналов в таблице
   - Добавление новых каналов
   - Удаление каналов
   - Редактирование настроек каналов

2. **Поиск**

   - Поиск по названию канала
   - Поиск по профилю пользователя
   - Поиск по ID канала
   - Отображение сообщения "Ничего не найдено" при отсутствии результатов

3. **QR-коды**

   - Генерация QR-кодов для авторизации
   - Отображение инструкций для пользователя

4. **Модальные окна**
   - Окно для добавления нового канала
   - Окно для редактирования настроек канала

## Структура проекта

```
telegram-channel-manager/
├── index.html          # Основная HTML-страница
├── style.css           # Стили и адаптивность

├── uiManager.js        # Управление интерфейсом
├── eventManager.js     # Обработка событий
├── main.js             # Инициализация и запуск приложения
└── README.md           # Документация проекта
```

## Как запустить проект

1. Создайте папку для проекта
2. Сохраните все файлы в соответствующие папки
3. Откройте файл `index.html` в любом браузере
